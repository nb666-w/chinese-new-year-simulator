# 🧨 过年回家模拟器 (Chinese New Year Simulator) v5.0

> "七大姑八大姨的灵魂拷问，你能扛过几轮？"

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**过年回家模拟器** 是一款基于生成式 AI (LLM) 的像素风文字冒险游戏。玩家扮演一名在大城市打拼的年轻人，春节回家面对各路亲戚的“关心”与“拷问”。

游戏不仅仅是简单的对话，还包含了一个**动态的八卦生态系统**和**Roguelite 养成机制**。你的每一次回答都会被亲戚们记录、传播，并影响未来的剧情走向。

## 🌟 核心特色 (Features)

*   **🤖 真实 AI 对话**: 接入 DeepSeek / Gemini / GLM 等大模型，亲戚们的反应不再是预设脚本，而是根据你的回答实时生成的真实反馈。
*   **🕸️ 八卦生态系统 (Gossip System)**: 亲戚之间互通声气。你告诉大姑的薪资，二舅马上就会知道。小心你的谎言被当场拆穿！
*   **🧬 世代传承 (Roguelite)**: 即使今年“破防”了也没关系。游戏结束后会根据表现结算【阅历值】，用于解锁【厚脸皮】、【大心脏】等永久天赋，助你来世再战。
*   **📓 情报收集**: 偷听亲戚的秘密（把柄），在关键时刻抛出【绝杀】选项，让对方瞬间破防。
*   **🎨 像素画风**: 精致的像素艺术背景与动态立绘，营造浓厚的过年氛围。

## 🎮 如何开始 (How to Play)

### 在线体验 (Demo)
[https://nb666-w.github.io/chinese-new-year-simulator](https://nb666-w.github.io/chinese-new-year-simulator)

### 本地运行 (Local)
1.  克隆本项目到本地：
    ```bash
    git clone https://github.com/nb666-w/chinese-new-year-simulator.git
    ```
2.  直接双击打开 `index.html` 即可运行（无后端依赖，纯前端实现）。

### 关于 API Key
由于游戏依赖大模型 API 生成对话，你需要自行准备 API Key。支持以下服务商：
*   **Google Gemini** (免费/付费)
*   **DeepSeek**
*   **通义千问 (Qwen)**
*   **智谱 AI (GLM)**
*   **Moonshot (Kimi)**

*注意：API Key 仅包含在你本地浏览器中，不会上传到任何服务器。*

## 🛠️ 技术栈 (Tech Stack)

*   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
*   **Pixel Art**: Generated by AI & Hand-tuned
*   **AI Integration**: Direct Fetch API calls to LLM endpoints
